# hotel_reservation_app

A Flutter project for learning and experimenting with hotel reservation features using Firebase.

Features

List and detail pages for hotels

Select check-in/check-out dates

Choose number of adults and children with age selection

Hotel image gallery with tap-to-zoom

Firestore integration for hotels, rooms, and bookings

Firebase Authentication for user login

## Getting Started

Requirements

Flutter 3.13+

Android Studio / VS Code

Optional: Firebase account to connect your own Firestore & Auth

Installing

Clone the repo and get dependencies:

git clone <your-repo-link>
cd hotel_reservation_app
flutter pub get

Firebase Setup

Android: android/app/google-services.json

iOS: ios/Runner/GoogleService-Info.plist

These are demo configs. To use your own Firebase project:

Create Firebase project

Enable Firestore and Authentication

Replace config files and update lib/main.dart if needed

Running the App
flutter run


Test on Android or iOS emulator or a real device.
